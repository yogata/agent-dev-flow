## 目的

機械的テキスト置換や複数ディレクトリ横断の是正を含む PR で、置換漏れ、回帰を完了宣言前に検出するため、再 grep 0 件証拠を完了条件として必須化する。
既存の mechanical-replacement-rules.md「再現性の担保」節に手順が存在するが運用未徹底であった事象（PR #1090 で 3 件残存、PR #1122 で回帰修正）を、品質ゲート（QG-3/QG-4）と PR 本文記載で構造的に担保する。

## 要件

| ID | 要件 |
|---|---|
| REQ-0153-001 | 機械的テキスト置換または複数ディレクトリ横断の是正を含む PR は、置換対象パターンの再 grep 結果が 0 件であることを完了証拠とすること |
| REQ-0153-002 | 機械横断是正の完了証拠（再 grep 結果 0 件）は PR 本文に記載すること |

## 適用範囲

- **対象**: 機械的テキスト置換、複数ディレクトリ横断是正を含む PR の完了証明、QG-3/QG-4 の機械横断是正向け検査項目、mechanical-replacement-rules.md「再現性の担保」節の必須手順化、PR 本文への再 grep 結果記載
- **対象外**: 機械的置換の対象文字種、パターン一覧の詳細（mechanical-replacement-rules.md）、QG-3/QG-4 検査項目の具体的文案（agentdev-quality-gates skill references）、QG-1〜QG-4 主ゲート体系の変更
