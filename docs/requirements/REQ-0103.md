## 目的

Command / Skill / Template / Script の責務境界を明確にし、重複記述・責務混線・参照不整合を防止する。

## 要件

| ID | 要件 |
|---|---|
| REQ-0103-001 | command はユーザー向け入口、入出力、ガードレール、高レベル手順を定義すること |
| REQ-0103-002 | command は共通判断基準や大きな状態機械を直接所有せず、skill を参照すること |
