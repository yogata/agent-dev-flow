## 目的

Command / Skill / Template / Script の責務境界は明確であり、重複記述・責務混線・参照不整合が発生しない。

## 要件

## 適用範囲

- **対象**: command / skill / template / script の責務境界、AgentDevFlow namespace、ドメイン状態ディレクトリ、skill naming / frontmatter / progressive disclosure、runtime-only 配布制約、repo-local command / skill の責務境界、共通処理の配置境界、Skill 粒度・参照妥当性・内部構造参照境界、inspect-skills の検査対象を直接修正しない診断責務、完了報告 template の責務境界
- **対象外**: source/projection・sync・repo type・consumer install の配布基盤要件、`.agentdev/drafts/` と draft type registry の配置要件、inspect ドメイン状態と inspect command lifecycle の要件、個別 command の詳細手順、テストケース本文、`.sisyphus/` 運用、package manager / marketplace plugin としての配布方式
