## 目的

`agentdev-workflow-lifecycle` が宣言的定義のみを提供するスキルとして REQ-0103-095 に完全準拠するよう、同 Skill の references/ に残存する5つのコマンド固有手続きファイルを目的別スキルへ移管する。これにより SKILL.md の「宣言的定義のみ。手順・手続きは含まない」宣言と実体の矛盾を解消し、5コマンド（case-open, intake-from-github, intake-promote, backlog-review, inspect-docs）の DO NOT USE FOR 衝突参照を解決する。

本要件は REQ-0103-095（workflow-lifecycle 責務限定）の実装手段を定義する。REQ-0103-095 は WHAT（何を除外するか）を定義済みであり、本要件は WHERE（どこへ移管するか）を定義する。
