## 目的

Runtime command（`src/opencode/commands/agentdev/*.md`）は、実行骨格（Step / Guardrail）に直接寄与しない補助参照記述を持たない構成であり、実行時トークン消費を抑える。command を薄い dispatcher として保つ方針（REQ-0119-001）を満たす。

## 要件

| ID | 要件 |
|---|---|
| REQ-0120-001 | runtime command の一次対象を `src/opencode/commands/agentdev/*.md`（13 command）とする |
| REQ-0120-002 | runtime command は、Step または Guardrail の条件文内で直接参照されていない独立した補助参照節を含まない |
| REQ-0120-003 | Step または Guardrail の条件文内で直接参照されている記述は保持する |
| REQ-0120-005 | 高頻度 command（`case-run.md`, `case-close.md`, `req-save.md`）が本要件の優先対象である |
| REQ-0120-006 | runtime command は Step / Guardrail の実行骨格が欠損しない構成を維持すること |
| REQ-0120-007 | `docs/` や `skills/` は本要件の対象外である |

## 適用範囲

- **対象**: `src/opencode/commands/agentdev/*.md` の参照節構成、高頻度 command 優先、Step/Guardrail 実行骨格の完全性
- **対象外**: `docs/specs/` 全体の説明量、skill 本文や `references/` の全面簡略化、`.opencode/commands/`（projection）、新しい command 構造
