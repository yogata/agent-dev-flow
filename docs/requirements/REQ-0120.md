## 目的

Runtime command（`src/opencode/commands/agentdev/*.md`）の実行時トークン消費を最適化するため、実行骨格（Step / Guardrail）に直接寄与しない補助参照記述を除去する。command を薄い dispatcher として保つ方針（REQ-0119-001）に適合させる。

## 要件

| ID | 要件 |
|---|---|
| REQ-0120-001 | runtime command の一次対象を `src/opencode/commands/agentdev/*.md`（13 command）とする |
| REQ-0120-002 | Step または Guardrail の条件文内で直接参照されていない独立した補助参照節を削除対象とする |
| REQ-0120-003 | Step または Guardrail の条件文内で直接参照されている記述は保持する |
| REQ-0120-004 | `case-run.md` の `## Manual References`（lines 127-135）を削除する。各 skill の load タイミング情報は対応する Step 内に既に参照されているため、独立節としての保持は不要 |
| REQ-0120-005 | 高頻度 command（`case-run.md`, `case-close.md`, `req-save.md`）を優先適用対象とする |
| REQ-0120-006 | 削除後に Step / Guardrail の実行骨格が欠損していないことを確認する手順を含む |
| REQ-0120-007 | `docs/` や `skills/` への即時一括適用は対象外とする |

## 適用範囲

- **対象**: `src/opencode/commands/agentdev/*.md` の非必須参照節、高頻度 command 優先、Step/Guardrail 欠損確認手順
- **対象外**: `docs/specs/` 全体の説明量最適化、skill 本文や `references/` の全面的な簡略化、`.opencode/commands/` への直接適用、新しい command 構造の導入

## 関連ドキュメント更新候補

- `src/opencode/skills/agentdev-command-authoring/references/command-authoring-standards.md` — token 最適化基準の追記候補

## Requirement Source

- runtime command から非必須参照記述を除去する
