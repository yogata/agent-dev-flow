## 目的

AgentDevFlow 基準文書・検査ルール・runtime 定義から RFC2119 規範語と使用ルールを完全に削除し、自然文ベースの必達要件・禁止事項の表現へ再基準化する。

## 要件

| ID | 要件 |
|---|---|
| REQ-0122-001 | active REQ/SPEC/ADR から RFC2119 使用ルールを完全に削除する |
| REQ-0122-002 | active REQ/SPEC/ADR から RFC2119 規範語（SHALL/MUST/SHOULD/MAY/SHALL NOT/MUST NOT/SHOULD NOT/MAY NOT）の記載を完全に削除する |
| REQ-0122-003 | RFC2119 前提の記述を自然文ベースの必達要件・禁止事項の表現へ再基準化する |
| REQ-0122-004 | integrity 検査から RFC2119 marker 検査および RFC2119 前提の判定定義を除去または新基準へ置換する |
| REQ-0122-005 | runtime skill/command/reference/test に残る RFC2119 依存記述を追従修正する |
| REQ-0122-006 | 削除後、横断検索で active 対象範囲に RFC2119 関連記載の残存がないことを確認する |

## 適用範囲

- **対象**: active REQ (0101-0121), SPEC, ADR, integrity, runtime
- **対象外**: retired, 履歴, 実装コード振る舞い
