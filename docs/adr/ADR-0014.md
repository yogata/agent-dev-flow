# ADR-0014: ADR / SPEC 再分類基準

## 決定

- **SPEC**: 「現在の構造・契約・ルール」を記述する。agent-dev-flow レポジトリ専用
- **ADR**: 「なぜその決定をしたか」の記録
- **Guides**: 人間向けナビゲーション層。規範的権限を持たない

本判断は ADR-0017（文書種別責務境界）に統合・再定義された。

## 関連する決定

- ADR-0017: 文書種別責務境界（本判断の後継）
- ADR-0013: runtime/authoring 関心分離（本判断の基盤）
