# ADR-0015: docs/specs 非runtime依存宣言

## 決定

docs/specs/ は agent-dev-flow レポジトリ専用の管理文書であり、runtime 配布物の依存先ではない。runtime 配布物は自己完結する。

本判断は ADR-0018（runtime 独立性）に統合された。
