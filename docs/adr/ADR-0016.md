# ADR-0016: skill references runtime-only 制約

## 決定

skill references/ は runtime 配布物のみを含める。authoring-only 資料は references/ に含めない。

本判断は ADR-0018（runtime 独立性）に統合された。

- ADR-0013: runtime/authoring 関心分離（本判断の基盤）
