# RU: `/repo/integrity-check` ローカル化に伴う docs 修正

## 背景

`/agentdev/integrity-check` および関連スキルは、以下の repo-local command / skill に置き換え済みである。

- `.opencode/commands/repo/integrity-check.md`
- `.opencode/skills/repo-agentdev-integrity/*.md`

この変更により、integrity-check は consumer project に配布される `/agentdev/*` コマンドではなく、agent-dev-flow repository 自体を検査する repo-local command として扱う。

## 問題

`docs/` 配下に、旧 `/agentdev/integrity-check`、旧 `agentdev-integrity`、旧 `sync-opencode.ps1`、または consumer project でも integrity-check を使えるように読める記載が残っている。

単純な名称置換で済む箇所と、repo-local / consumer runtime の責務境界に合わせた意味修正が必要な箇所が混在している。

## Source Summary

チャットで、以下の修正対象と方針を合意した。

- `docs/specs/runtime-package-boundary.md`
- `docs/guides/consumer-project-setup.md`
- `docs/specs/integrity-rule-catalog.md`
- `docs/specs/artifact-contracts.md`
- `docs/requirements/REQ-0108.md`
- `docs/specs/integrity-contracts.md`
- `docs/guides/README.md`
- `docs/adr/ADR-0020.md`

主な修正観点は以下である。

- `/agentdev/integrity-check` を現行機能として案内しない。
- 現行 self-hosting repo 検査は `/repo/integrity-check` / `repo-agentdev-integrity` として記載する。
- consumer project 向け文書では `/repo/integrity-check` を配布コマンドとして案内しない。
- consumer project の drift 検出・修復は consumer 用 script に寄せる。
- 旧 root `sync-opencode.ps1` を現行 script として案内しない。

## 統合理由

対象箇所はいずれも、`/repo/integrity-check` の repo-local 化と、consumer runtime への配布境界に関する同一変更に属する。

個別修正として分割すると、docs 内で repo-local / consumer runtime の責務境界が不整合になりやすい。したがって、1つの Requirement Unit として統合し、文書横断で同じ意味に揃える。

## 要件化の方向

`docs/` 配下の旧記載を、以下の方針で修正する。

### `/agentdev/integrity-check`

現行コマンドを指す場合は `/repo/integrity-check` に置換する。

consumer project に配布されるコマンドとして読める箇所では、単純置換せず、consumer 用 script に書き換える。

### `agentdev-integrity`

現行スキルを指す場合は `repo-agentdev-integrity` に置換する。

旧構成を説明する履歴文脈では、旧 `agentdev-integrity` を残してよい。

### `.agentdev/vocabulary-registry.md`

現行配置を指す場合は `.opencode/skills/repo-agentdev-integrity/references/vocabulary-registry.md` に置換する。

### `sync-opencode.ps1`

self-hosting sync を指す場合は `scripts/sync-self-opencode.ps1` に置換する。

consumer install / check を指す場合は、用途に応じて以下へ置換する。

- `scripts/install-consumer-opencode.ps1`
- `scripts/check-consumer-opencode.ps1`
