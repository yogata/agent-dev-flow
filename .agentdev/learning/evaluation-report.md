### 問題クラス2: Windows環境でのgh CLI出力エンコーディング問題

- **根本原因**: Windows (PowerShell 7) 環境で gh CLI の UTF-8 出力を pwsh パイプライン・リダイレクト・Write tool 経由で処理すると文字化けが発生する。pwsh の stdout エンコーディング変換、BOM付きUTF-8 出力、`>` リダイレクトの encoding 処理に問題がある
- **再発条件**: Windows環境で gh CLI の JSON出力・ファイル入出力を PowerShell 経由で扱う場合
