# integrity-rule-catalog.md 行964 の保護対象リストが陳腐化

## 観測内容

`docs/specs/integrity-rule-catalog.md` 行964 の IR-044 exemption 条件の回帰テスト設計において「保護対象の真陽性: REQ-0114-082、REQ-0144-008」と記載されている。
Issue #1109 の是正により両行から SPEC 詳細（Step 番号、件数、"fixture" キーワード）が除去され、両行は真陽性ではなくなった。よって行964 の記述は陳腐化している。

## 影響

現状の行964 は「保護対象の真陽性」として存在しない状態を指している。
RU-0011 実施前に同箇所を根拠にテストを設計すると、既に是正済みの行を保護対象として扱い、テストの前提が崩れる可能性がある。

## 課題

陳腐化した保護対象リストを更新し、是正済み経緯を明記する。

## 既存要件との関連

- Issue #1109、PR #1119（merged, squash 61c98d50）
- REQ-0114-082 / REQ-0144-008（本 PR で是正済み・非真陽性化）

## 対応方針の方向性

- 行964 の更新を RU-0011（検出ロジック改良）のスコープに含めるか、本件から個別 Issue を起こして先行是正するか
- 保護対象リストから REQ-0114-082 / REQ-0144-008 を除去し、両行が「是正済み（非真陽性化）」となった経緯を併記する
