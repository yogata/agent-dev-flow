# 一文一行機械判定の残課題（不完全文・述語欠落・体言止め混在）

## 観察

PR #1091（OU-004「一文一行是正 - 複数文1行を分割」）は 7 ディレクトリ・407 件の機械的分割を実施し、複数文 1 行問題の大部分を解消した。ただし完了条件の厳密版（不完全文・述語欠落・体言止めと述語あり文の混在）は「個別判断が必要なため本 PR では機械的分割のみ対応」と明示して残置した。

PR #1088（中黒）、#1089（em-dash）、#1090（LLM 表現）でも同様に、「この機械判定ルール自体を SPEC または skill references に明文化すると再現性が上がる（本 PR のスコープ外）」という共通の残置表現が繰り返されている。

## 修正されなかった理由

不完全文・述語欠落・体言止め混在は文脈判断が必要で、機械的検出が困難。本 PR では確実に機械判定できる「複数 `。` を含む行の分割」のみ対応した。

## 課題

- 不完全文・述語欠落を検出する機械判定アルゴリズムの実現可能性（形態素解析等の導入必要性）
- 体言止めと述語あり文の混在判定基準の明文化
- 上記アルゴリズムを `mechanical-replacement-rules.md` または japanese-tech-writing skill references のどちらへ位置付けるか
- 残存箇所の網羅的検出（別途 `inspect-docs` または独立 Issue での対応）
